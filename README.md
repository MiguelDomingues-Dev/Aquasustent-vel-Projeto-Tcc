# AquaSustentável - Projeto TCC

**Projeto em desenvolvimento do TCC - Curso Técnico em Informática**

## Sobre o Projeto
O **AquaSustentável** é uma iniciativa voltada para a sustentabilidade e a conscientização ambiental, com o objetivo de reduzir o consumo excessivo de água. O projeto propõe uma solução tecnológica que permite aos cidadãos monitorarem e controlarem seu uso de água de forma eficiente.

Por meio de ferramentas de análise e controle, os usuários serão capazes de:
- Visualizar o consumo de água em tempo real.
- Definir limites de consumo.
- Tomar decisões mais conscientes e sustentáveis.

Dessa forma, o projeto promove a redução de desperdício e a gestão inteligente dos recursos hídricos, com aplicações em áreas rurais e urbanas.

## Tecnologias Utilizadas
O projeto está sendo desenvolvido com as seguintes tecnologias:

### Hardware:
- **Arduino** (com **ESP32**): Para coleta e envio de dados.
- **Sensor de Fluxo de Água**: Para medição do consumo.
- **Relé e Válvula Solenóide**: Para controle do fluxo de água.

### Software:
- **HTML, CSS e JavaScript**: Para desenvolvimento do site responsivo.
- **Frameworks e Bibliotecas Gráficas**: Para exibição de dados em gráficos e tabelas interativas.
- **Comunicação via Wi-Fi**: Para integração do hardware com a plataforma web.

## Funcionalidades Principais
- Monitoramento em tempo real do consumo de água.
- Controle remoto do fluxo de água através de um sistema web.
- Interface intuitiva com relatórios dinâmicos.
- Incentivo ao uso consciente e à sustentabilidade.

## Objetivos Alinhados ao ODS 12
Este projeto está em conformidade com os **Objetivos de Desenvolvimento Sustentável (ODS)**, especialmente o **ODS 12**: **Consumo e Produção Responsáveis**, promovendo a redução do desperdício de água e a otimização do uso desse recurso vital.
